1.1. How do I create a hidden file or folder? Name of a file or a directory should start with a dot character. To create a “hidden” directory you should use command “mkdir”, for example: $mkdir “.myHiddenDirectory”To create “hidden” file you could use “touch” command, for example: $touch “.myHiddenFile”1.2. How do I display it in the CLI?You could use command “ls” with a argument “-a” (show all files), for example:$ls -a2. How do I create multiple nested directories, like /c/Users/myusername/these/folders/are/just/for/fun?You should use command “mkdir” with an argument “-p” (Create intermediate directories as required), for example:$mkdir -p “these/folders/are/just/for/fun” 3. How do I append a message to a file, without a newline character, so the output would be…There are many ways to do that, for instance, you could use “echo” command with an argument “-n” (Do not print the trailing newline character) like that:$echo -n “first message\nsecond message\n”>>file.txtAnd there are many more, for instance “printf” command, etc...
